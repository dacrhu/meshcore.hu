# Terminal Chat CLI

:::info Forrás
Fordítás alatt — az eredeti: [Terminal Chat CLI](https://docs.meshcore.io/terminal_chat_cli/) (MIT licenc).
:::

A Terminal Chat egy parancssori kliens, amellyel telefonos alkalmazás nélkül,
számítógépről lehet a meshen üzenni.

✍️ *Kidolgozandó: telepítés, kapcsolódás, parancsok.*
