# nRF52 energiagazdálkodás

:::info Forrás
Fordítás alatt — az eredeti: [nRF52 Power Management](https://docs.meshcore.io/nrf52_power_management/) (MIT licenc).
:::

Az nRF52-alapú eszközök (pl. RAK4631, T1000-E, Seeed XIAO) fejlett
energiakezelése: mélylemerülés elleni védelem, feszültség-alapú ébresztés,
leállási okok naplózása.

✍️ *Kidolgozandó szakaszok:*

- funkciók (boot-feszültségvédelem, voltage wake, shutdown-okok)
- támogatott eszközök
- technikai részletek
- kapcsolódó CLI-parancsok
