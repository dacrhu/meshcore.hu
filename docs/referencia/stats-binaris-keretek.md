# Statisztika bináris keretek

:::info Forrás
Fordítás alatt — az eredeti: [Stats Binary Frame Structures](https://docs.meshcore.io/stats_binary_frames/) (MIT licenc).
:::

A statisztika-parancsok (core, rádió, csomagszámlálók) bináris kereteinek
felépítése, példakóddal.

✍️ *Kidolgozandó szakaszok:*

- parancs- és válaszkódok, altípusok
- keretszerkezetek (C/C++ struktúrapéldák)
- lekérdezés- és feldolgozáspéldák (Python, JavaScript)
