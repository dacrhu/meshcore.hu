# Companion protokoll

:::info Forrás
Fordítás alatt — az eredeti: [Companion Protocol](https://docs.meshcore.io/companion_protocol/) (MIT licenc).
:::

Ezt a protokollt használja a MeshCore alkalmazás (és bármely saját fejlesztésű
kliens) a Companion firmware-t futtató eszközzel való kommunikációra BLE-n,
USB-n vagy Wi-Fi-n.

✍️ *Kidolgozandó szakaszok:*

- hivatalos programkönyvtárak
- biztonsági megjegyzés
- BLE-kapcsolat (service-ek, karakterisztikák, MTU, parancssorrend)
- csomagszerkezet
- parancsok és válaszok
- csatornakezelés
