# Payload-formátum

:::info Forrás
Fordítás alatt — az eredeti: [Payload Format](https://docs.meshcore.io/payloads/) (MIT licenc).
:::

A [csomagformátumra](csomagformatum.md) épülő payload-típusok referenciája.

✍️ *Kidolgozandó szakaszok:*

- alapfogalmak
- csomópont-hirdetés (advert)
- nyugtázás (acknowledgement)
- visszaadott útvonal, kérés, válasz, szöveges üzenet
- anonim kérések (room server / repeater login stb.)
- csoportos szöveges üzenet és datagram
- vezérlőadatok (control data)
