# KISS modem protokoll

:::info Forrás
Fordítás alatt — az eredeti: [MeshCore KISS Modem Protocol](https://docs.meshcore.io/kiss_modem_protocol/) (MIT licenc).
:::

A KISS modem firmware a MeshCore-eszközt szabványos KISS TNC-ként teszi
használhatóvá soros porton.

✍️ *Kidolgozandó szakaszok:*

- soros port beállításai
- keretformátum és type byte
- szabványos KISS-parancsok (host ↔ TNC)
- SetHardware kiterjesztések, hibakódok, események
- adatformátumok (rádióparaméterek, statisztikák, akku stb.)
