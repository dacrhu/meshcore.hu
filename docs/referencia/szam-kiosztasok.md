# Szám-kiosztások

:::info Forrás
Fordítás alatt — az eredeti: [Number Allocations](https://docs.meshcore.io/number_allocations/) (MIT licenc).
:::

A protokollban használt lefoglalt számértékek (típuskódok, csoport-adattípusok)
nyilvántartása — saját alkalmazás fejlesztésekor innen kell azonosítót
választani.

✍️ *Kidolgozandó.*
